Maven wrapper
===

Сборка
---
`$./build.sh`

По умолчанию в качестве базового образа берется тег `latest`
Для сборки необходимой версии передать переменную окружения BASE_TAG, в которой указать нужный тег

```$ BASE_TAG=1.8.0 ./build.sh```

Или

```$ export BASE_TAG=1.8.0```

```$ ./build.sh```
